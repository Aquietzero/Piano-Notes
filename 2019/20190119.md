# 20190119

练琴2小时

今天辗转了两个地方练琴，但还是保证时间了，真心不容易。还有两天就艺考结束，到时候就没有那么多人了。

周末上课，今天花了很多时间在贝奏上面。换了几个琴，尝试了不同的踏板，感觉不同的琴差异极为巨大。有的在音阶部分踩上踏板刚刚好，有的则非常糊。回想起家里的琴，好像在哪个地方踩都很糊，如果是这样的话，那我在家是难以练到踏板的。

今晚还是重点练习发展部，速度上去之后这个地方很容易错位，右手尝试用更平缓的姿势触键，尽量用指肚触键，配合手腕的摇摆，但是右手5指的sf不容易做，目前最多就只是突出一下这个音而已。

巴赫的前奏曲继续单手练习，既然双手无法练习就只能通过单手继续寻找快速下舒服的感觉了。其实左手有些34指的部分也是不容易的，专门抽了那些地方出来单独练习。
